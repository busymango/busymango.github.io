v2ray科学上网

发表于 2019-10-20 | 分类于 黑科技

V2ray是一个融合了各种翻墙协议的集成性软件。 是Shadowsocks 之后非常好用的代理软件，但是由于 V2Ray 的配置略复杂，GUI 客户端不完善，所以 V2Ray 并没有像 Shadowsocks 在科学上网人群之中那么流行。不过，有了一键安装脚本之后，这一切就变得简单了。v2ray一键安装脚本 & 管理脚本支持 V2ray 绝大多数传输协议，WebSocket + TLS，HTTP/2，Shadowsocks，动态端口，集成 BBR 和锐速优化等。

本文参考链接：搬瓦工vps被墙，使用v2ray复活被墙vps

一、注册一个域名

到阿里云或者其他注册商，注册一个xyz域名xxx.xyz，首年6元，用一年不续费，再注册就是，很简单。

二、cloudflare域名设置

  1. 注册cloudflare账号， 注意要通过邮箱验证 。

  2. cloudflare添加域名。

  3. 点击Add site进入下一步，选择free免费套餐。

  4. 继续下一步到添加域名解析记录，选择A记录，name随便你自己填写比如：bwg，IPv4填写你的搬瓦工VPS的IP，云朵请点成灰色仅dns解析，先不使用CDN（很重要，切记！)。

  5. 继续下一步，提示修改Nameservers（域名服务器），并提供了两个域名服务器地址，打开域名注册商的域名管理，修改成新的DNS服务器，修改域名服务器一般10分钟左右生效。

  6. cloudflare SSL设置，SSL设置为Full。

三、搬瓦工一键搭建v2ray

  1. 下载Xshell : https://www.netsarang.com/zh/free-for-home-school/

  2. 进入搬瓦工KiviM，先stop当前IP，然后点击Install new OS，将系统切换到Centos7，记住更改后的密码。

  3. Xshell连接到VPS服务器

  • 连接 将VPS服务器IP、端口信息输入，协议为SSH

  • 用户身份验证 用户名为root，密码为上步骤操作得到的密码。

  1. 在Xshell中输入命令
  1. 一键安装脚本运行后，依据提示选择：

  • 安装或卸载 1.安装

  • 请选择V2Ray传输协议 4. WebSocket + TLS

  • 选择V2Ray端口 回车默认

  • 输入域名 bwg.xxx.xyz(cloudflare记录名.域名)

  • 确认解析正确 cmd命令窗口输入: ping bwg.xxx.xyz看ip是否为VPS的ip，是就ping通了

  • 是否自动配置TLS y

  • 是否开启网站伪装和路径分流 默认回车

  • 是否开启广告拦截 默认回车

  • 是否配置shadowsocks 因为要用shadowrocket所以选择配置

  • 安装配置信息

安装成功后会输出V2Ray和shadowsocks配置信息

  1. 现在可以输入命令v2ray来管理了。v2ray url 生成vmess url链接，复制保存好vmess url链接，配置客户端的时候需要用到。
  • 管理脚本命令如下：

v2ray info 查看 V2Ray 配置信息
v2ray config 修改 V2Ray 配置v2ray url 生成vmess url链接
v2ray link 生成 V2Ray 配置文件链接
v2ray infolink 生成 V2Ray 配置信息链接
v2ray qr 生成 V2Ray 配置二维码链接
v2ray ss 修改 Shadowsocks 配置
v2ray ssinfo 查看 Shadowsocks 配置信息
v2ray ssqr 生成 Shadowsocks 配置二维码链接
v2ray status 查看 V2Ray 运行状态
v2ray start 启动 V2Ray
v2ray stop 停止 V2Ray
v2ray restart 重启 V2Ray
v2ray log 查看 V2Ray 运行日志
v2ray update 更新 V2Ray
v2ray update.sh 更新 V2Ray 管理脚本
v2ray uninstall 卸载 V2Ray

四、V2Ray客户端

在使用客户端之前，很重要的一步，到cloudflare你的域名解析处把云朵点成橙色，使用CDN，很重要！

V2RayN配置使用，复制之前那保存下来的vmess url链接，然后从剪贴板批量导入URL即可

v2ray windows客户端：https://github.com/2dust/v2rayN/releases

v2ray 安卓客户端：同样的导入vmess url链接或者扫描配置二维码就可以了。

v2ray IOS客户端：美区下载shadowrocket

v2ray mac os客户端：https://github.com/Cenmrev/V2RayX/releases


我在搭建过程中遇到的问题：

  • 阿里云域名申请后必须通过实名认证，状态显示为正常才可以使用。
  • cloudflare账号注册必须通过邮箱验证，修改DNS后的域名才能ping通过。
  • cmd中ping命令无效，通过 我的电脑–>属性–>高级系统设置–>环境变量–>path 将cmd地址添加进去。
  • ping 的域名应该是cloudflare记录名.阿里云域名。
  • 客户端运行之后网速极其慢，等待一段时间后问题得到解决，但使用速度还是慢。
